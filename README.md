# free stuff every developer should know about

A list of tools, APIs, and resources that I personally use or have found useful. Started keeping this when I got tired of bookmarking random things and losing them.

Feel free to add things via PR — only rule is it has to actually be free (not "free trial" stuff).

---

## AI tools

| Tool | What it does | Notes |
|------|-------------|-------|
| [Google AI Studio](https://aistudio.google.com) | Gemini API, free tier | Best free option right now |

| [Hugging Face](https://huggingface.co) | Open source models | Huge library |
